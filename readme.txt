This repository contains R code (RMarkdown document birdsfirefrequency1.Rmd) and data for the Bayesian multispecies occupancy model presented in:

Michael J. M. Franklin (1,3), Richard E. Major (2), Michael Bedward (1), and Ross A. Bradstock (1).
'Relative avian mobility linked to use of fire-affected resources in forested landscapes'.
Accepted for publication in Forest Ecology and Management on 23/06/2021.

2) Australian Museum Research Institute, Australian Museum, 1 William Street, Sydney, 2010, Australia
3) Corresponding author. Email: mjmf080@uowmail.edu.au
